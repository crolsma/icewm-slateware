= IceWM Slateware Theme

A gray and teal theme with minimal aesthetics.
The Makefile includes default values for the font and colors,
which can be easily changed from the CLI.

image::screenshot.png[]

== Installation

`make [XDG_CONFIG_HOME=path] [attribute=value...]`

Possible attributes are:

* `fontname`
* `fontsize`
* `background`
* `foreground`
* `highlight`
* `lowlight`
* `button`
* `monitor`

Consult the Makefile for default values.
